Регистрация перехода права
================

Тип Документооборота : *rightmovement_request*

Описание: *Подача заявлений в электронном виде в части регистрации перехода права*

Создание документооборота осуществляется методом  ``POST /docflows/{docflowType}``.
    
    *  гед docflowType  (тип документооборота) - rightmovement_request

В тело метода должен быть помещен json запроса, в котором содержатся необходимые данные для инициализации документооборота.
Json запроса состоит из:
    * requestId - Опциональный внешний идентификатор запроса клиента.
    * options – Параметры документооборота, которые определяют необходимую информацию для отправки запроса в Росреестр. Параметры (options) документооборота RightmovementRequestPost включают в себя следующие данные:
       
        * rightMovementType: основание для перехода права *на данный момент доступен тип sale*
        * object: сведения об объекте недвижимости (:doc:`objects/object` )
        * seller: сведения о продавце. Для конкретизации сведений о продавце необходимо определить тип собственности продаваемого объекта: 
        
            * individualOwnership  Описание продавца если он владеет объектом недвижимости единолично включает

                * owner: Описание собственника :doc:`objects/docflowmember`

            * cooperativeOwnership: Описание продавцов если объект недвижимости находится в совместной собственности включает: 

                * spouse1:  описание свдедений о супруге :doc:`objects/docflowmember`
                * spouse2:  описание свдедений о супруге :doc:`objects/docflowmember`
            
            * sharedOwnership:Описание продавцов если объект недвижимости находится в долевом владение включает
            
                * shares описание каждого дольщика ( :doc:`objects/share` [] )
            
            * evidenceOfOwnership Если право собственности продавца возникшло до 1998 г и не было зарегистрированно необходимо приложить набор правоустанавливающи документов: 
            
                * documents: набор прикладываемых документов для данного рег действия (:doc:`objects/applied-document`[])
.. Hint::
    Регистрация права обязательна для совершения сделки. Росреестр предлагает зарегистрировать права собственника на объект единовременно с переходом права. Также оплата пошлины за данное регистрацинное действие не взымается 
-------

        *  buyer:  Сведения о покупателе. сведения о продавце. Для конкретизации сведений о покупателе необходимо определить тип собственности в которую приобретается объект: 
            
            * individualOwnership  Описание продавца если он приобретает объект недвижимости в единоличное владение

                * owner: Описание  будующего собственника  :doc:`objects/docflowmember`
                * mortgage: Если приобретение недвижимости осуществляется с помощью ипотеки в силу закона, необходимо приложить набор документов :doc:`objects/mortgage`
            
            * cooperativeOwnership  Описание продавцов если они приобретают объект недвижимости в совместное владение

                * spouse1: Описание  будующего собственника супруга  :doc:`objects/docflowmember`
                * spouse2: Описание  будующего собственника супруга  :doc:`objects/docflowmember`
                * mortgage: Если приобретение недвижимости осуществляется с помощью ипотеки в силу закона, необходимо приложить набор документов :doc:`objects/mortgage`
                * marriageCertificate:  Свидетельство о браке :doc:`objects/applied-document`

.. Note::
    Для подтверждения факта браке необходимо предоставить Свидетельство о браке
-------
                
            * sharedOwnership: Описание продавцов если они приобретают объект недвижимости в долевое владение
           
                * shares: описание каждого дольщика :doc:`objects/share` [] 
       
       * appliedDocuments: Описание прилагаемых документов для регистрации перехода права

            * contractOfSale: обязательным документом для регистрации продажи является договор купли продажи (:doc:`objects/applied-document`)
            * other: Другие документы  :doc:`objects/applied-document`
